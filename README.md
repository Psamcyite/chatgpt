## Features

- **Chat Interface**: Users can interact with the chatbot in a user-friendly interface.
- **Real-time Responses**: Chatbot provides real-time responses to user inputs.
- **Customizable Appearance**: Interface is customizable using Tailwind CSS utility classes.
- **Type Safety**: Developed using TypeScript for type safety and improved development experience.

---

## Technologies Used

- [React](https://reactjs.org): JavaScript library for building user interfaces.
- [Vite](https://vitejs.dev): Next-generation frontend build tool that provides a faster and leaner development experience.
- [Tailwind CSS](https://tailwindcss.com): Utility-first CSS framework for designing responsive web applications.
- [TypeScript](https://www.typescriptlang.org): Superset of JavaScript that adds static types to the language.

## Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/Psamcyite/chatgpt-clone.git
   cd chatgpt-clone
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```
